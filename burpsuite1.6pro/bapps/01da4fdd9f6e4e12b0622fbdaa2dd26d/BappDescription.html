This extension enables Pcap files to be imported into the Burp Target site map, and passively scanned. It can be used in cases where an HTTP client does not support proxying but it would be useful to scan, inspect or replay the HTTP traffic using Burp.

The extension makes use of jNetPcap, which relies on a native binary to be loaded by Java. Download the correct binary for your platform and copy to your java.library.path (e.g. your %PATH% on Windows):

  • Windows: x32 x64
  • Linux: x32 x64

The jNetPcap native binary requires a packet capture tool to be installed. If you do not already have this installed, you can download it from the following locations:

  • Windows: WinPcap
  • Linux: libpcap

After installation, an "Open Pcap file..." option will be added to the context menu on the trees under the Scanner and Target tabs.

Requires Java version 7.
